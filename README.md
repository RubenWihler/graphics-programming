# graphics-programming

## Introduction

Ce dépôt retrace mon aventure dans l'univers de la programmation graphique. L'objectif final est de créer une simulation de particules à l'aide des compute shaders d'OpenGL.

Ce projet est avant tout personnel et a pour but de m'initier à un paradigme complètement nouveau pour moi. Pour maximiser mon apprentissage, je documenterai soigneusement chaque étape et expérience tout au long de ce parcours.

N'ayant que très peu d'expérience dans ce domaine, je m'attends à rencontrer de nombreuses difficultés. Cependant, je suis motivé à relever ce défi et à surmonter ces obstacles !

## Documents et structure

Ce dépôt est organisé autour de plusieurs mini-projets qui m'ont permis d'acquérir les bases de la programmation graphique. Chaque projet dispose de son propre dossier, contenant un fichier README.md détaillant ses objectifs et expliquant comment le compiler.

Tous les mini-projets sont regroupés dans le dossier projects.

La documentation joue un rôle clé dans ce projet. Le dossier doc contient un rapport général qui synthétise les différentes étapes de mon apprentissage. Ce rapport sert de fil conducteur entre les projets, offrant une vue d'ensemble de mon parcours sans entrer dans les détails techniques.

Les aspects techniques spécifiques à chaque mini-projet sont documentés dans des rapports individuels situés dans les dossiers correspondants.
